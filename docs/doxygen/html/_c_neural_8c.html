int epochs )

Trains a neural network using the specified parameters.

Weighted sums get forward propagated (forward pass) by calculating linear combinations applying an activation function for each node in each layer. After all the training examples go through 1 epoch, the loss and gradient are calculated based on the optimizer. Neural network parameters are adjusted with the learning rate accordingly.

Parameters
nnneural network type
numLabelsnumber of labels (training examples)
inputsarray of inputs (features)
labelsarray of labels
lossFunctiona string corresponding to a loss function
optimizera string corresponding to an optimizer
learningRatelearning rate to be applied in gradient descent
epochsnumber of epochs (forward passes through the whole dataset)
